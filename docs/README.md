## Project Name
Веселый ипподром
## Description 
Проект представляет собой информационную систему для автоматизации деятельности ипподрома. Система предназначена для учета лошадей, жокеев, проведения забегов, а также формирования отчетов для различных категорий пользователей: управляющего ипподрома, жокеев и владельцев лошадей.
